starfruit: ascension

a game made in 48 hours for indie speed run 2013

click to play »

MISSION

Go as high as you can!

RULES

  1. Stars duplicate themselves every once in a while.
  2. If a star hits anything other than a cloud, it explodes.
  3. If a star hits another star, they both explode.
  4. If all the stars explode, GAME OVER.

CONTROLS

action effect
click left attract stars
press space pause
press F go fullscreen
